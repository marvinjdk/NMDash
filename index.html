⚡ Norse NerdMiner Dashboard

Live-status view for your ESP32 NerdMiner fleet. Works on GitHub Pages with optional local status proxy.

ℹ️ How this works

  • This page can be hosted directly on GitHub Pages.
  • Due to browser security, GitHub Pages cannot directly poll your local miners.
  • To get true live data, you can run a small local proxy that outputs a JSON summary.
  • If no proxy is detected, the dashboard shows offline/demo mode.
